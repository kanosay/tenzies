#Tenzies
###This is a simple dice game built using React. The goal of the game is to roll dice until all dice show the same value. You can freeze dice by clicking on them. The timer counts down from 120 seconds, and you can reset the game whenever you win.
![image](https://github.com/user-attachments/assets/8644784d-5f34-48f1-a2ef-3fb6e555681d)
## How to run

- cd tenzies
- npm install
- npm run dev
